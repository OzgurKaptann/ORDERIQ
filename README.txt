# OrderIQ

OrderIQ is a self-serve SaaS platform for small and medium fast-service food businesses.

Core capability:
- Create digital menu
- Generate QR
- Accept customer orders
- Manage orders in kitchen
- View basic analytics

## V1 Scope

Included:
- Auth & tenant
- Onboarding
- Menu builder
- QR menu
- Ordering
- Kitchen screen
- Basic analytics

Excluded:
- Payments
- Integrations
- Advanced analytics
- Multi-branch UI
- AI / recommendations

## Tech Stack

- Backend: FastAPI
- DB: PostgreSQL
- ORM: SQLAlchemy
- Frontend: Next.js
- Infra: Docker Compose

## Goal

A merchant can:
- register
- setup menu in minutes
- generate QR
- receive orders
- manage kitchen
